= 03 • Installing Proxmox VE

== BIOS tweaks (HP EliteDesk)

. F10 → System Options  
. **Disable Secure Boot**  
. **Uncheck “Configure Storage Controller for Intel Optane”** → forces AHCI  
. Video memory down to 32 MB (saves RAM)
